# Sentiment Analysis of Flipkart Product Reviews

## Objective
This project aims to classify Flipkart customer reviews as Positive or Negative
using Natural Language Processing and Machine Learning techniques.

## Dataset
The dataset was provided by Innomatics Research Labs and contains customer reviews
scraped from Flipkart by the Data Engineering team. No additional scraping was performed.

## Methodology
- Text Cleaning & Normalization
- TF-IDF Feature Extraction
- Logistic Regression Model
- Evaluation using F1-Score

## Model Performance
- F1-Score: ~0.95

## Web Application
A Streamlit web application was developed to perform real-time sentiment analysis
on user-provided reviews.

## Deployment
The application is deployed on AWS EC2 (Free Tier).

## Technologies Used
- Python
- Pandas, Scikit-learn, NLTK
- Streamlit
- AWS EC2
